Front-end technology

HTML (HyperText Markup Language)

The backbone of any webpage. Used to structure content on the web (e.g., headings, paragraphs, images, forms).

CSS (Cascading Style Sheets)

Used for animations, responsive designs, and visual presentation.

JavaScript

Adds interactivity and dynamic behavior to websites. Enables functionalities like form validation, sliders, and interactive maps.
